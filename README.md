# 📊 Student Performance Analysis

# 📌 Project Overview

This project focuses on analyzing student performance data to uncover key factors that influence academic outcomes. Using data analysis techniques, the project identifies trends, patterns, and insights that can help improve student success.

# 🎯 Objectives

Analyze student performance based on various factors

Identify relationships between study habits and scores

Discover trends in academic performance

Provide actionable insights for better decision-making

# 📈 Key Insights

1) Students whose parents have higher education level genrally score better. 

2) Parent education shows positive correction. 

3) Students with standard lunch tend to perform better than those with free lunch.

4) Students who completed test preparation score higher than other student 

5) Female students tend to out perform male student score higher in math 

6) Group E scored higher than other groups
